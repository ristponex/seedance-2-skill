# Seedance 2.0 Video Skill

This is an AI agent skill for generating videos using ByteDance's Seedance models via Atlas Cloud API.

## API Details
- Endpoint: POST https://api.atlascloud.ai/v1/prediction/{prediction_id}
- Auth: Bearer token via ATLAS_API_KEY
- Current model: Seedance v1.5 Pro ($0.222/video)
- Coming soon: Seedance 2.0

## Usage
Generate videos from text prompts or reference images.
